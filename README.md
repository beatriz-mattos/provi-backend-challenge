# Provi backend challenge
Challenge desenvolvido com NodeJS + Express para a empresa Provi.

## Documentação:
https://documenter.getpostman.com/view/11226481/TVReeB49

## Stack utilizada:
- Axios;
- Bcryptjs;
- Date-fns;
- Dotenv;
- Express;
- Jsonwebtoken;
- Knex;
- MySQL;
- Ts-node-dev;
- 100% Typescript;
- Uuid
